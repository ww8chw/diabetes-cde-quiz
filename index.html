① 口服降血糖藥 — 機轉 / 低血糖 / 體重
藥物類別代表藥主要機轉單用低血糖體重
雙胍類 BiguanideMetformin抑制肝臟糖質新生（第一線）否中性／略降
磺醯尿素 SUglimepiride, gliclazide, glipizide促胰島素分泌是增加
美格列奈 Meglitiniderepaglinide, nateglinide促胰島素分泌（餐後）是增加
TZD 胰島素增敏劑pioglitazonePPARγ 增加胰島素敏感性否增加＋水腫
α-葡萄糖苷酶抑制劑acarbose延緩雙醣→單醣吸收，降餐後否中性
DPP-4 抑制劑sitagliptin提高腸泌素濃度（口服）否中性
GLP-1 RAliraglutide…（針劑）腸泌素促效，抑食慾、緩胃排空否減少
SGLT2 抑制劑—腎臟排糖否減少
  • 會低血糖：只有「促胰島素分泌劑」＝ SU + Meglitinide（與胰島素）。
  • 會增重：SU、Meglitinide、TZD、胰島素。不增重／減重：Metformin、acarbose、DPP-4、GLP-1、SGLT2、orlistat。
  • SU 與 Meglitinide 機轉相同 → 不宜併用。
  • Metformin：抑制肝糖新生、改善血脂、影響 維生素 B12（非 B1）；肝腎不全／老年慎用以防乳酸中毒。UKPDS 證實對心血管有益。
  • Acarbose 造成的低血糖 → 須補單醣（葡萄糖／蜂蜜），吃飯、牛奶無效。
  • TZD（pioglitazone）：水腫、體重增加、血容比下降、心衰惡化、骨折風險。
  • 心血管實證：GLP-1 RA、SGLT2i 有保護；DPP-4i 為中性。
  • 腸泌素 incretin＝GLP-1（遠端 L 細胞）＋GIP（近端 K 細胞），進食後上升、半衰期極短被 DPP-4 分解。
② 胰島素 — 種類與藥動學
類型代表外觀高峰用途
速效aspart, lispro澄清有餐時，控餐後（依飯後2hr調量），可餐前/後即打
短效Regular (RI)澄清有餐前30–60分注射
中效NPH混濁有(2–4hr起效)基礎
長效glargine, detemir澄清無基礎，每天固定時間
超長效degludec (Tresiba)澄清無(t½≈25hr)基礎，多六聚體緩釋
  • 基礎胰島素：NPH、glargine、detemir、degludec。不可當基礎：Regular、aspart、lispro（餐時）。
  • Glargine：澄清、無高峰、≈24hr、每天固定同一時間、PH≈4 不可靜脈注射、不可與他種混合。
  • 胰島素幫浦（CSII）只用速效胰島素；離身過久即可能高血糖／DKA（離身4hr不影響為錯）。
  • 速效 vs RI：RI 需餐前30–60分、易餐前低血糖；速效可即時注射、夜間低血糖較少、降 A1C 略佳。
  • 降 A1C 能力最強＝胰島素（無劑量上限）；acarbose 最弱。
  • 劑量計算：胰島素需求量 = 醣量 ÷（每單位可涵蓋醣克數）。ICR 法則：500÷TDD（速效）或 450÷TDD。
  • 懷孕中後期胰島素阻抗↑，需求約增至孕前2倍（≈0.9–1.2 甚至1.8–2.0 u/kg/天）。
  • 積極治療＝每日多次注射或幫浦（DCCT 證實併發症較少）。
③ 胰島素注射技術
  • 吸收速率：腹部最快 ＞ 上臂 ＞ 大腿 ＞ 臀部最慢。
  • 注射部位需輪替、避開脂肪增生(lipohypertrophy)／發炎／水腫，注射前洗手、部位乾淨乾燥。
  • 筆針一針一換、單次使用；重複使用會鈍化、增加疼痛與脂肪增生。
  • 減痛：胰島素回溫至室溫、排除空針氣泡、酒精乾後再扎。
  • RI＋NPH 預混抽好之注射器冷藏穩定 ≤ 1 週。
④ 糖尿病神經病變
  • 最常見＝遠端對稱性多發性神經病變（足部麻木、感覺遲鈍；大纖維影響觸覺/振動覺）。
  • 影響感覺、運動、自主神經三者皆可。
  • 篩檢時機：第2型確診當下即篩；第1型診斷後約5年起，之後至少每年一次。
  • 單一神經病變(mononeuropathy)多為急性、自限性會自行緩解。
  • 近端運動型（糖尿病肌萎縮）主要侵犯運動神經 → 近端肌無力。
  • 自主神經：姿勢性低血壓、運動耐受不良、胃輕癱（上消化道）、便秘/腹瀉/失禁（下消化道）、無感低血糖、性功能/膀胱障礙。
  • 高血糖經 AGE 破壞髓鞘加重病變（非保護）；嚴格血糖控制可降約60%發生率(DCCT)。
⑤ 糖尿病足 — PEDIS / Wagner / 篩檢
  • PEDIS：Perfusion 灌流 / Extent 範圍 / Depth 深度 / Infection 感染 / Sensation 感覺。
  • Wagner 分級：0～5 級，共 6 級。
  • 足潰瘍最主要危險因子＝周邊神經病變（保護性感覺喪失）。
  • 篩檢工具：10g(5.07) 單股尼龍纖維→觸壓覺；128 cps 音叉 / Rydel-Seiffer→振動覺；觸診足背動脈→血管。
  • 周邊血管病變可因合併神經病變而無痛，不能因不痛而排除。
  • 壞疽：濕性＝感染＋持續壞死；乾性未必須截肢。
  • 抗血小板：clopidogrel、cilostazol、ticagrelor；Rivaroxaban 是抗凝血(Xa)非抗血小板。
  • 清創後不建議立即直接縫合；乾淨傷口避免用優碘（細胞毒性）；可用負壓引流、保持生長因子環境。
  • 鎚狀/爪狀趾 → 穿深、寬鞋頭的鞋保護腳趾、留活動空間。
⑥ 傷口營養
  • 促癒合營養素：維生素 A、C、鐵、Omega-3、鋅(≈30mg/天×12週)、充足水分、蛋白質/胺基酸。
  • 精胺酸(Arginine)＝一氧化氮前驅物 → 擴張血管、增加灌流、促進癒合。
  • 維生素 A：抗發炎、調節免疫、促上皮修復。
  • 熱量建議（壓瘡）≈ 30–35 大卡/公斤/天；先滿足總熱量再補蛋白質。

⑦ 糖尿病分類與診斷標準

分類

類型機轉／特徵重點
第 1 型胰島 β 細胞被自體免疫破壞，絕對缺乏胰島素自體抗體 ICA、IAA、GAD、IA-2；與 HLA-DR3/DR4 相關；好發兒童青少年；易酮酸中毒；須終生胰島素
第 2 型胰島素阻抗＋相對分泌不足占 90%↑；與肥胖、家族史、年齡有關；多無酮體；起初可口服藥
妊娠糖尿病 GDM懷孕中後期首次發現的高血糖產後須重新分類；日後得 T2DM 風險高
其他特異型MODY（單基因）、胰臟疾病、藥物（類固醇）、內分泌疾病—

診斷標準（符合任一項，無症狀者須改日複驗）

  • HbA1c ≥ 6.5%
  • 空腹血漿血糖 FPG ≥ 126 mg/dL（禁食至少 8 小時）
  • OGTT 第 2 小時血糖 ≥ 200 mg/dL（75 克葡萄糖）
  • 典型高血糖症狀（多吃多喝多尿、體重減輕）＋隨機血糖 ≥ 200 mg/dL

糖尿病前期（高危險群）

  • 空腹血糖偏高 IFG：FPG 100–125 mg/dL（部分舊版 110–125）。
  • 葡萄糖耐受不良 IGT：OGTT 2 小時 140–199 mg/dL。
  • HbA1c 5.7–6.4%。

新陳代謝症候群（台灣標準，≥ 3 項即成立）

  • 腹部肥胖：腰圍 男 ≥ 90cm／女 ≥ 80cm
  • 三酸甘油酯 TG ≥ 150 mg/dL
  • 高密度脂蛋白 HDL：男 < 40／女 < 50 mg/dL
  • 血壓 ≥ 130/85 mmHg
  • 空腹血糖 ≥ 100 mg/dL

流行病學

  • 糖尿病為國人十大死因之一；心血管疾病是糖尿病患首要死因。
  • 約有相當比例患者未被診斷（無症狀期長）。

  • 升糖指數 GI：以含 50 克醣類食物比較血糖上升幅度；高 GI 食物使血糖快速上升。
  • 酒精：男性 ≤ 2 當量、女性 ≤ 1 當量／天，且須隨餐（空腹飲酒易低血糖）。
  • 減重：每日減少 500–1000 大卡，每週減 0.5–1 公斤；極低熱量飲食 VLCD ≤ 800 大卡須醫療監督。
  • 醣類計算（carbohydrate counting）用於彈性胰島素治療配對醣量。
⑩ 運動治療（FITT）
  • 頻率 F：每週 ≥ 3–5 次，勿連續 2 天以上不運動。
  • 強度 I：中等強度（運動時心跳較安靜增加約 20–30 次／分，能講話不能唱歌）。
  • 時間 T：每次 20–60 分鐘；每週 ≥ 150 分鐘中等強度有氧。
  • 型態 T：有氧為主，搭配阻力訓練。
  • 禁忌：血糖 > 250–300 mg/dL 併酮體陽性時不宜運動（恐加重高血糖／酮酸）。
  • 運動可能造成運動後（延遲性）低血糖，達 12–24 小時；使用胰島素／促泌劑者運動前後須監測、備糖。
  • 有周邊神經病變者避免承重型運動以防足部潰瘍，宜選游泳、騎車。
⑪ 戒菸與行為改變
  • 吸菸增加心血管與微血管併發症風險；菸量越大風險越高。
  • 跨理論模式行為改變五階段：思考前期 → 思考期 → 準備期 → 行動期 → 維持期。
  • 賦能（Empowerment）：以病人為中心，協助其自我決策與自我管理，而非單向命令。
  • 衛教技巧：示範教學、回覆示教（teach-back）、設定可達成的小目標。
⑫ 血糖監測 SMBG / 糖化血色素
  • SMBG 自我血糖監測：即時反映當下血糖，利於調整飲食、運動、藥物。
  • HbA1c：反映 2–3 個月平均血糖；建議每 3 個月追蹤一次（穩定者可半年）。受溶血、貧血、血紅素變異影響。
  • 糖化白蛋白／果糖胺：反映 2–3 週短期血糖。
  • 連續血糖監測 CGM：評估血糖變異與目標範圍時間 TIR。
⑬ 急性併發症：低血糖 / DKA / HHS

低血糖

  • 定義：血糖 < 70（臨床常用 < 60）mg/dL。症狀：冒冷汗、發抖、心悸、飢餓、意識改變。
  • 15 法則：清醒者口服 15 克單醣（葡萄糖、果汁、方糖），15 分鐘後復測，未改善再給一次。
  • 意識不清／無法吞嚥：升糖素 glucagon 1 mg 肌肉注射或靜脈葡萄糖，勿強行餵食。
  • Whipple's triad：低血糖症狀＋當下血糖低＋給糖後緩解。

⑭ 慢性併發症：眼 / 腎 / 神經 / 大血管

視網膜病變

  • 非增殖性 NPDR：微血管瘤、出血、硬性滲出物；增殖性 PDR：新生血管（可致玻璃體出血、視網膜剝離）。
  • 第 2 型確診時即須眼底檢查，之後每年追蹤；血糖、血壓控制可延緩。

腎病變

  • 早期指標：微量白蛋白尿（UACR 30–300 mg/g 或 30–300 mg/天）。
  • 首選藥物：ACEI／ARB（兼具腎臟保護）；嚴格控制血壓 < 130/80。
  • SGLT2 抑制劑具腎臟保護效果。

神經病變

  • 最常見為遠端對稱性多發性神經病變（手套襪套式麻木、刺痛、感覺喪失）。
  • 篩檢：10 克 Semmes-Weinstein 單股纖維（5.07）、音叉震動覺、MNSI 量表。
  • 自主神經病變：姿勢性低血壓、胃輕癱、無症狀低血糖。

大血管病變

  • 冠心病風險為一般人 2–4 倍；周邊動脈疾病以 ABI（踝肱指數）評估，間歇性跛行。
  • 控制三高、戒菸、抗血小板治療。
⑮ 共病管理：高血壓 / 血脂
  • 血壓目標 < 130/80 mmHg；首選 ACEI／ARB（尤其合併蛋白尿）。
  • 血脂目標：LDL < 100（高危險 <70）、HDL 男 >40／女 >50、TG <150。
  • 降血脂首選 statin（HMG-CoA 還原酶抑制劑）；TG 極高（>500）防胰臟炎可用 fibrate。
  • 建議地中海飲食、減鈉、規律運動、戒菸併行。
⑯ 特殊族群：懷孕 / 老年 / 兒童 / 手術

妊娠糖尿病 GDM

  • 篩檢時機：懷孕 24–28 週。兩步驟：50 克篩檢 1 小時 ≥ 140 → 再做 100 克 3 小時 OGTT。
  • 100 克 OGTT 切點（Carpenter-Coustan）：空腹 95／1hr 180／2hr 155／3hr 140，≥ 2 項異常即診斷。
  • 孕期血糖控制首選胰島素（多數口服藥不建議）；產後 6–12 週重新分類。

老年

  • 目標放寬（HbA1c 7.5–8.5%視健康狀況），首重避免低血糖；慎用長效 SU。

兒童／青少年

  • 以第 1 型為主；初期可有「蜜月期」胰島素需求暫降（≤0.5 U/kg）。

手術期

  • 圍術期目標血糖約 140–180（避免 >200 及低血糖）；可用 GKI 或靜脈胰島素控制。
⑰ 衛教 / 個案管理 / 照護團隊
  • 糖尿病共同照護網：以個案管理師（衛教師）為核心，結合醫師、護理、營養、藥師等跨專業團隊，提供持續性整合照護。
  • 糖尿病衛教五大核心：飲食、運動、藥物、監測、併發症預防（自我照顧足部）。
  • 糖尿病護照／登錄：記錄檢驗值與追蹤，落實定期回診與品質指標（HbA1c、眼底、腎功能、足檢）。
  • 生活品質評估工具：WHOQOL、SF-36、DQOL。
  • 足部自我照護衛教：每日檢查、勿赤腳、溫水洗腳、修平趾甲、合腳鞋襪、勿自行處理雞眼。
